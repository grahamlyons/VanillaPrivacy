/**
* Additional privacy settings for version 2 of the Vanilla Forums software.
*
* This was tested with version 2.0 rc_1 of Vanilla.
* 
*/

Add this folder into the 'plugins' folder in the root of the Vanilla2 installation 
and the enable plugin via the 'Plugins' section of the dashboard.

Access to 'Activity' and 'Profiles' can then be controlled via the 'Users & Roles'
section.
